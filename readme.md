# Alterra Test

This project is a mini project playground created by [Stefan](https://www.linkedin.com/in/stefanuscc/), inspired by alterra test to complete these 2 problems :
- Palindrome Check for String
- Diagonal matriks calculation

## Available Scripts

In the project directory, you can run:

### `go test alterra`
